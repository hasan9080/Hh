Básicos do Kubernetes

Este tutorial fornece um guia sobre os básicos do sistema de orquestração de clusters Kubernetes. Cada módulo contém algumas informações de fundo sobre as principais funcionalidades e conceitos do Kubernetes, e um tutorial para seguir.

Usando os tutoriais, pode aprender a:

  • Implementar uma aplicação conteinerizada num cluster.
  • Escalar a implementação.
  • Atualizar a aplicação conteinerizada com uma nova versão do software.
  • Depurar a aplicação conteinerizada.

O que o Kubernetes pode fazer por si?

Com os serviços web modernos, os utilizadores esperam que as aplicações estejam disponíveis 24/7, e os desenvolvedores esperam poder implementar novas versões dessas aplicações várias vezes ao dia. A conteinerização ajuda a empacotar o software para servir estes objetivos, permitindo que as aplicações sejam lançadas e atualizadas sem tempo de inatividade. O Kubernetes ajuda a garantir que essas aplicações conteinerizadas corram onde e quando quiser, e ajuda-as a encontrar os recursos e ferramentas de que precisam para trabalhar. O Kubernetes é uma plataforma de código aberto pronta para produção, desenhada com a experiência acumulada da Google em orquestração de contentores, combinada com as melhores ideias da comunidade.
